# 任务一规划（deprecated）
- 第1步：空间降维
    - 目标：`(n_time, N_features)` -> `(n_time, latent_dim)`
    - 方法：对于每个时期（Belief or Perspective），分别对每个被试、每个脑区的每个`trial`，在空间上做PCA/VAE
    - 输出：20个被试 x 2个时期 x 4个脑区，每个都包含一整套 `trial` 数据
- 第2步：时序压缩
    - 目标：`(n_time, latent_dim)` -> `(1, embed_dim)`
    - 方法：被试内。使用LSTM, ST-GCN, 或SimCLR等方法学习一个编码器，将每个 `trial` 的数据压缩成一个向量
    - 输出：每个被试、每个时期、每个脑区，都有一组*试次原型向量*（将每个试次的原型向量叠到一起）（`N_trials` x `embed_dim`）
- 第3步：条件原型向量（聚合）
    - 目标：`(N_trials, embed_dim)` -> `(8, embed_dim)`
    - 方法：被试内。将第2步得到的试次向量进行分组，然后取平均值
    - 输出：每个被试、每个时期、每个脑区，都有 8个*条件原型向量*（形状 `(8, embed_dim)`）。    
- 第4步：跨被试对齐
    - 目标：对齐21个被试的 `(8, embed_dim)` 空间
    - 方法：现在对齐比较好，因为不在高维体素上做，维度降下来了。使用Procrustes分析或SRM对齐空间
    - 输出：一组代表群体共性的*条件原型向量* `(8, embed_dim)` （每个时期的每个脑区一组）
- 第5步：RDM生成与LLM对齐
    - 目标：`(8, embed_dim)` -> `(8, 8)`，然后与LLM比较。
    - 方法：
        1. 使用第4步得到的8个向量，计算它们之间的 cosine 距离、euclidean 距离，或训练 `SVM` 解码器得到混淆矩阵，即最终 `fMRI_RDM` (8x8矩阵)
        2. 将这个矩阵与项目提供的 `LLM_rdm` 进行斯皮尔曼相关等后续分析
# 思路
应该先做特征提取（被试内 or 跨被试）（这样可以提高信噪比）

然后，被试内：

做空间降维（实际上特征提取已经进行了一些降维，可以再用PCA/VAE压一压，还能起到统一维度的作用）

然后做时序压缩（方法A：ST-GCN，LSTM；方法B：直接取平均；方法C：SimCLR）
- A：数据太少，跑不动
- B：稳健，但很蠢。但是，考虑到这个任务只有4+6个时间点，应该也没逝吧
- C：必须在B的基础上做。见下

最后生成矩阵，将矩阵跨被试平均起来

注意：不能把两个时期（belief, perspective）的时间拼接起来，要分别去做decoding（因为它们中间存在时间间隔，而且属于两个不同的任务，这样可以去比较LLM的哪个部分的功能与哪个时期的fMRI特征更像）。同样，也不能把四个脑区的数据拼接起来。

方法、名词详解：
- LSTM：用于时序压缩。将每个时间点的向量一个个喂进去，拿到最后一个时间点的输出
- ST-GCN：将当前时间点的每个特征当做节点（即，把整体看成图如何随着时间变化的问题）
	- 输入：
	    1. 数据 (X)： 一个试次的（PCA降维后的）数据，形状 `(n_time, N_features)`
	    2. 图 (A)：一个邻接矩阵，形状 `(N_features, N_features)`
	- 问题：A从哪里来？
		- 在fMRI分析中，这个“图”通常是**功能连接 (Functional Connectivity)** 矩阵。你必须先计算你的所有PC在所有时间点上的相关性矩阵，才能把它作为“图”喂给 ST-GCN
	- 输出：
		- 一个单一的原型向量，形状 `(1, embed_dim)`
- SimCLR：
	- 它是一个训练框架，一种自监督学习方法
	- 它的目标是训练一个encoder，能够做到：同一事物的不同视角，其向量表征应该相似；而不同事物的向量表征应该不相似
	- 具体方法：
		- 取一个原型向量$X$，进行两次不同的、随机的数据增强（例如，加一点噪声、随机遮挡一部分），分别得到$X_{i},X_{j}$，组成一个正样本对
		- encoder学习如何拉近$X_i$和$X_j$的表征
		- 该批次的所有其他的样本，称为负样本
		- encoder学习如何推开$X_i$和所有负样本 ($Y$, $Z$...) 的表征
	- 训练完成后，使用这个encoder，就能将原来的向量映射到另一个向量，从而将不同条件的向量表征分开
- MVPA (Multi-Voxel Pattern Analysis)，即多体素模式分析，是一种fMRI的分析框架或哲学，而不是一个单一的具体算法
	- 它假设信息是分布式的，编码在数千个体素的协同模式中，而不是单个体素里
	- 流程：
		- 特征提取
		- decode：训练一个decoder，如SVM，获得混淆矩阵
	- 所以，可以当这个名词没什么用，因为整套流程就是遵循这个哲学的

## 关于体素选择
我们的讨论始终围绕一个核心矛盾：
1. **被试间差异（批次效应）** 极大，这被你的“孤岛”UMAP图所证实。
2. **单个被试的数据量** 极小（例如，每个时期~64个试次）。
这导致了两种截然相反的特征选择哲学：
### 1. 哲学一：“被试内”特征选择 (Per-Subject)
这种方法承认“批次效应”是不可逾越的鸿沟，因此**必须**在每个被试内部、隔离地进行所有操作。这是你学长建议的“比较decoding matrix”所隐含的标准路径。
#### 选项 A: 单变量过滤 (Univariate Filtering)
- **方法：** 标准fMRI GLM（质量单变量） + ANOVA F-test (`f_classif`)。
- **原理：**
    1. **独立运行：** 为每个体素单独运行一个统计模型（GLM），看这个体素的激活值是否随8个条件显著变化。
    2. **过滤：** 它为每个体素计算一个F-score。我们保留分数最高的若干个体素。
- **优点：** 简单、快速；在数据量太少的情况下统计上最鲁棒（最不容易过拟合）。
- **缺点：** 这种方法不关心体素间的“综合信息”。它无法捕捉到需要多个体素协同编码的信息。
#### 选项 B: 多变量选择 (Multivariate Selection)
- **方法：** SVM-RFE（递归特征消除）或 Lasso (GLM+L1)。
- **原理：**
    1. **综合运行：** 在一个被试的~64个样本上，运行一个包含所有体素的多变量模型（`条件 ~ W1*Voxel1 + ... + WN*VoxelN`）。
    2. **选择：** SVM-RFE会抛弃那些对于SVM来说不重要的体素。Lasso会将它认为无用的体素权重设为0，从而选出特征。
- **优点：** 理论上更优，因为它确实在寻找能综合预测条件的体素组合。
- **缺点：** 在 `N=64` 个样本上训练一个拥有大量特征的模型，存在**极高**的过拟合风险。
### 2. 哲学二：“跨被试”特征选择 (Group-Level)
这是你提出的、理论上更优越的路径。它基于一个关键假设：一个有监督的模型（其目标是预测8个条件）应该能学会忽略与条件无关的噪声（即被试差异）。
- **方法：** **使用L1正则化的多项逻辑回归 (GLM + L1 / Lasso)**。
- **原理：**
    1. 被试内标准化：首先对每个被试的数据进行Z-score标准化，以消除基线差异（为每个体素，独立地，沿着样本/时间轴计算均值和标准差）
    2. **聚合：** 将所有被试（`N_total = 64 * 20`）的数据聚合起来。
    3. **训练一个模型：** 在这个巨大的聚合数据集上，训练一个L1正则化的多项逻辑回归（`Y = 8个条件`, `X = 5000+个体素`）。
    4. **选择：** L1正则化项会自动将那些在所有被试中不能一致预测8个条件的体素权重设为0。
- **优点：**
    1. **数据量大：** 正如你指出的，模型在海量数据上训练，更可能学到鲁棒的特征。
    2. **解决对齐：** 它提供了一个统一的特征空间（例如500个“组通用”体素），一劳永逸地解决了“跨被试不可比”的问题。
- **缺点：**
    1. **信噪比风险：** 此方法赌的是“8个条件”的信号足够强，能够在巨大的（即使是标准化的）被试间噪声中被模型捕捉到。如果信号太弱，Lasso可能什么也选不出来。
- 同理，自然可以尝试SVM-RFE。

## 大致流程
为每一个被试的每一个脑区分别训练一个独立的降维模型 / 独立使用方法
1. 数据聚合
    - 遍历该被试、该脑区下的所有条件（`False_Belief`, `True_Belief`）及所有试次（`trial_00` ~ `trial_31`）
    - 将每一个试次的 `Signal` 矩阵（形状为 `n_time, N_features`） 提取出来，并将它们在时间维度上拼接
2. 训练降维模型
    - 在训练前要对聚合数据进行标准化处理，即，计算整个聚合数据的均值和标准差，并用它们来归一化数据
    - 目的：
        1. 标准化确保了模型不会被信号强度的绝对差异所主导
        2. 统一训练确保了所有试次（无论来自何种条件）都将被投影到同一个潜在空间中
3. 转换试次数据
    - 使用训练好的模型对标准化后的试次矩阵进行转换
    - 该模型将原始的 `(n_time, N_features)` 矩阵映射为新的 `(n_time, latent_dim)` 矩阵
4. 产出与重复
    - 将上一步生成的所有低维矩阵（`n_time, latent_dim`）按照原始的数据结构（例如：`Left_PFC` -> `False_Belief` -> `trial_00` -> `Signal`）进行存储
    - 重复：将上述所有步骤在所有其他脑区和所有其他被试上重复执行
将得到一套与原始数据结构完全一致的H5文件，但其中所有的 `Signal` 数据都已从高维体素空间（`N_features`）被压缩到了低维潜在空间（`latent_dim`），同时保留了后续时序分析所需的 `n_time` 维度

另外实现了一套为所有被试使用同一降维方法的pipeline，可作为参考

# PCA/VAE/SimCLR 林乐天

> **注意：** 本节内容旨在纠正前文《任务一规划（deprecated）》中关于特征提取的重大理论误区。经实验验证，原定方案在处理低信噪比、高被试变异的 fMRI 数据时存在根本性缺陷。以下是对原方案的驳斥及修正后的技术路线定调。

### 1. 对原定“被试内独立 PCA/VAE”方案的驳斥
原方案提议：“*对于每个时期，分别对每个被试、每个脑区的每个 trial，在空间上做 PCA/VAE*”。这一设计存在致命的几何逻辑错误：

* **特征空间的正交性错位（Orthogonality Mismatch）**：
    对不同被试（如 TOM001 和 TOM002）独立进行 PCA，生成的特征向量（Principal Components）是基于各自数据的最大方差方向定义的。TOM001 的 `PC1` 可能代表“头动伪影”，而 TOM002 的 `PC1` 可能代表“视觉皮层响应”。
    **后果**：这导致不同被试生成的 Latent Space 在物理意义上完全不可比。强行将这些数学上不正交、语义上不对齐的向量进行跨被试聚合（Stacking/Averaging），等同于“将苹果的重量与橘子的颜色相加”，后续的 RDM 计算将毫无意义。

* **方差 $\neq$ 信号（The Variance Fallacy）**：
    PCA 和 VAE 均为无监督模型，其优化目标分别是最大化解释方差（Explained Variance）和最小化重构误差（Reconstruction Loss）。
    **现实**：在 fMRI 数据中，方差的主要来源是**生理噪声**（呼吸、心跳、血流动力学波动）和**被试特异性基线**，而非我们关注的心理理论（ToM）认知任务信号（Task Signal）。
    **结论**：在缺乏监督信号引导的情况下，PCA/VAE 会优先“去噪”掉微弱的任务信号，而保留高幅度的生理噪声。实验中 PCA 结果呈现的“一团云雾”和 VAE 呈现的“按被试聚类而非按任务聚类”证实了这一点。

### 2. 对“先平均后压缩”时序处理的驳斥
原方案提议：“*方法B：直接取平均... 方法C：SimCLR必须在B的基础上做*”。这是对对比学习机制的误解：

* **动态信息的丧失**：fMRI 的 BOLD 信号具有特定的血流动力学响应函数（HRF）时滞和形状。简单的时序平均（Averaging）会抹平这些包含认知加工过程的动态特征，导致信噪比进一步降低。
* **数据增强的缺失**：SimCLR 等对比学习框架的核心在于从“同一事物的不同视图（Views）”中学习不变性。如果预先将 Trial 平均为一个静态向量，模型就失去了利用时序波动（Temporal Jitter）作为自然数据增强的机会，无法学习到鲁棒的特征表征。

### 3. 修正后的技术路线：有监督对比学习（Supervised Contrastive Learning）

基于上述分析与实验验证，我们确立 **SimCLR (SupCon)** 为唯一有效的特征提取范式。其正确性在于：

* **流形对齐（Manifold Alignment）**：
    SimCLR 不是在每个被试内部闭门造车，而是在所有被试的数据上进行**联合训练（Joint Training）**。它迫使模型学习一个通用的投影函数，将不同被试的神经活动映射到同一个高维超球面上。
* **利用标签去噪（Label-guided Denoising）**：
    通过引入监督信号（Supervised Loss），模型被强制要求拉近属于同一 ToM 条件（如 `False Belief`）的样本，推远不同条件的样本。这相当于构建了一个**非线性过滤器**，主动抑制与任务无关的被试差异和生理噪声，从而提取出真正通用的神经表征。
* **几何结构的保留**：
    实验表明，SimCLR 提取的 Embeddings 不仅实现了类间分离（Separability），还保留了类内变异（Intra-class Variability）的丰富几何结构，这是后续 Task 2 进行表征相似性分析（RSA）并与大模型对齐的必要前提。

**最终结论**：放弃所有基于单被试的无监督降维尝试，全面转向基于群体数据的有监督对比学习框架。

## 关于特征选择

时间特征，由于血流伪影，我们选择4s/6s(/8s)的时刻进行研究，因为这个时间点的血流才是真正的看到题目后的神经活动变化。关于前面提到的选取高变脑区的提议，我也做了对应的试验，发现高变脑区实际上编码的是一些噪声相关的脑活动，不可用。另外，无论是SimCLR还是PCA还是VAE,都可以正确捕捉高变异的成分。所以没有必要做这个事情。
